What is it?

When flying in the X-Plane 11 flight simulator, I wanted a more precise readout of my propeller RPMs and manifold pressure. This is my solution, a representation of the Aerospace Logic MT200 in X-Plane. Written in C++, this plugin creates a floating window with the instrument. As you can see in picture two, the instrument displays the same information as the base instruments. It also uses the state of the master battery switch to determine if the instrument should be on or off. You can see what it looks like turned off vs on in pictures 3 and 4 respectively.

How does it work?

The plugin is built using the X-Plane SDK. The graphics in the window are drawn with OpenGL. I used bitmaps for both the background of the instrument and the font. These bitmaps are loaded in with LoadBMP.
