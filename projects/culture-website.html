What is it?

In a Literature class, we were anlyzing how external conflicts affect and change a culture. I choose to create a website to display my findings. This website has webquest-like elements where the viewer reads an article, then returns to the page to complete an activity (see picture 2).

How does it work?

The website is built with Bootstrap. Bootstrap is a useful css library that I mainly used for responsive rows and columns. This way, the site works on many different devices.
