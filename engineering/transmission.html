What is it?

This a 2 speed 3D printed transmission I designed in Fusion 360. It has a high, low, and neutral gear. It was designed to work with a small electric motor and servo. The motor would connect to the input shaft, and the servo would connect to the shift rod.

Versions

This is actually the second version I made, the first version is shown in picture 3. The main problem with the first version was its shift fork was too narrow and flimsy which caused it to break. It may have worked if made out of a stronger material, but since it is made out of PLA, it did not. The other major change I made in version two is the positioning of the input shaft. It is next to the output axle instead of in line with the output shaft.

How does it work?

As the input shaft turns, it turns the gears connected to it. The gears on the output axle are not connected to it, they turn freely. When the shift fork is moved, it slides a disc that connects the output axle to the freely spinning gears. Now, the output axle turns is connect to the input shaft via the selected gear ratio. This process works best when the system is moving so the shift fork and disc doesn't get stuck.
